# 🪟 Modern Glassmorphism Login Form

A sleek, semi-transparent login interface built with HTML5 and CSS3. This project demonstrates the **Glassmorphism** design trend, utilizing backdrop filters and subtle borders to create a "frosted glass" effect over a vibrant background.

## 🚀 Features
* **Glassmorphism UI:** High-end frosted glass effect using `backdrop-filter: blur(15px)`.
* **Modern Typography:** Clean look using the **Poppins** font family.
* **Icon Integration:** Professional input field appearance using **Boxicons**.
* **Fully Responsive:** Centered layout using Flexbox, compatible with all screen sizes.
* **Interactive States:** Smooth hover effects for links and the primary button.

## 🛠️ Built With
* **HTML5:** Semantic structure.
* **CSS3:** Custom properties, Flexbox, and transparency effects.
* **Boxicons CDN:** For the user and lock icons.
* **Google Fonts API:** For font embedding.
